\doxysubsection{Detailed Description}
A class responsible for handling multiple spreadsheets, including saving, loading, and managing them. 

The \doxylink{classSheetHandler}{Sheet\+Handler} class allows the user to add, save, load, and view spreadsheets. It also manages a menu for creating and running spreadsheets through user interaction.

\begin{DoxyNote}{Note}
The spreadsheets are stored in a specified directory (default is "{}sheets"{}). 
\end{DoxyNote}
